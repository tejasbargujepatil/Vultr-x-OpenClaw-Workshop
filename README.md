## 🚀 OpenClaw Setup

Follow all 6 steps **before arriving** on 31 May.

**Step 1 — Verify Node.js**
```bash
node --version   # Expected: v22.14.0 or above
npm --version    # Expected: 10.x or above
```

**Step 2 — Verify Docker & Git**
```bash
docker --version
docker run hello-world   # Expected: "Hello from Docker!"
git --version
```

**Step 3 — Install OpenClaw**

```bash
# macOS / Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

**Step 4 — Run the onboarding wizard** *(~2 minutes)*
```bash
openclaw onboard --install-daemon
```

**Step 5 — Verify the gateway**
```bash
openclaw gateway status
# Expected: Gateway listening on port 18789
```

**Step 6 — Open the dashboard**
```bash
openclaw dashboard
# Opens in your browser. If it loads — you're ready! ✅
```

> 💡 Send a message in the Control UI chat and get an AI reply — you're all set for the workshop! 🎉

### Vultr Deployment *(done during the workshop)*

```
1. Access  →  Vultr Marketplace — OpenClaw
2. Deploy  →  OpenClaw image on a Vultr instance
3. Config  →  Set up the server instance
4. Access  →  Dashboard + complete onboarding
5. Explore →  Deployment architecture with the facilitator
```

## 🏆 KubeCon Ticket — Winner Selection Guide

> [!IMPORTANT]
> For the **workshop speaker / session host only.**
> Announce the giveaway at the very start of the session — it drives real engagement throughout.

**Opening announcement script:**
> *"We have a KubeCon ticket to give away today — but it's not a lucky draw. We're looking for the most engaged, curious, and helpful person in the room. Stay active, ask good questions, help your neighbours, and post about it!"*

### Scorecard

| Criteria | Points |
|:---|:---:|
| ✅ Completed pre-setup before arriving | 1 |
| 🙋 Asked a thoughtful question during the session | 2 |
| 💡 Shared an interesting insight or real use case | 2 |
| 🤝 Helped a fellow participant during hands-on | 2 |
| 🚀 Successfully deployed OpenClaw on Vultr | 2 |
| 📢 Posted on LinkedIn live — tagged `@Vultr` + `#CloudNativeDayPune2026` | 3 |
| 🌟 Bonus: proposed a creative use case for OpenClaw | 1 |
| | **Max: 13 pts** |

### Running the Selection — Last 10 Minutes

```
Step 1  →  Ask who posted on LinkedIn (show of hands) — verify @Vultr tag + hashtag
Step 2  →  60-second volunteer huddle — align on top 1–2 candidates quietly
Step 3  →  Announce with a genuine reason: "KubeCon goes to [Name] because [reason]"
Step 4  →  Tiebreaker: "In one sentence — what would you build with OpenClaw?"
Step 5  →  Backup: crowd applause vote for 2–3 shortlisted names
Step 6  →  Photo immediately + LinkedIn post tagging winner + @Vultr
           Share KubeCon ticket details privately after the event
```
